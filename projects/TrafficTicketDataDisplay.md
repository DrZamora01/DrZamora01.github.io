In this project, I created a website that presents traffic ticket data using JSON files. The website offers two pages: one displaying all the available data and another allowing users to apply filters to view specific information. The website is built using JavaScript, Bootstrap for styling, Vue.js for data manipulation, and GitHub for hosting and easy navigation between pages.
